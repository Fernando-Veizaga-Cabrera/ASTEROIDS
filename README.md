# ASTEROIDS
Programacion Web - Juego Asteroids

Un juego clásico de Asteroids desarrollado con **JavaScript**, **HTML** y **CSS**, refactorizado bajo el patrón de diseño **Modelo-Vista-Controlador (MVC)**.

## 🕹️ Cómo jugar

* **Flecha Arriba:** Avazar la nave.
* **Flecha Izquierda/Derecha:** Rotar la nave.
* **Barra De Espacio:** Disparar.
* **Objetivo:** Destruye los meteoritos.

## 🛠️ Tecnologías

* **JavaScript (ES6+):** Lógica de clases orientada a objetos.
* **HTML5:** Para la estructura del juego.
* **CSS3:** Animaciones de texto y estilos de interfaz.
